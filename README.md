# NoSkillBoostFromToddlersBooksAndChildToys
Skills learned as children are not passed on to adult Sims.

This mod uses Mono Patcher Library by LazyDuchess. You also need to have that mod in your game. https://modthesims.info/d/687815/mono-patcher-library-0-3-0.html
Description:
Child skills don't transfer to adult.
Toddler's books don't transfer to adult skills.

Навыки, выученные в детстве не переходят взрослым симам.

Это мод использует библиотеки Mono Patcher автора LazyDuchess. Вы должны установить также этот мод. https://modthesims.info/d/687815/mono-patcher-library-0-3-0.html
Описание:
Детские навыки не трансформируются во взрослые.
Книги для тоддлеров не повышают навыки при взрослении.
